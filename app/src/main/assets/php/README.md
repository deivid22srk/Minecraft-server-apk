# PHP Binaries for Android

This directory will contain PHP binaries compiled for Android during CI/CD build.

- `arm64-v8a/` - PHP 8.4 for ARM64 devices (64-bit only)

These binaries are downloaded from: https://github.com/pmmp/PHP-Binaries

**Note:** Only ARM64 (64-bit) devices are supported. ARMv7 (32-bit) support has been discontinued.
